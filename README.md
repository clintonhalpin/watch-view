# watch-view
Prototype apple watch apps and preview on your phone! [Demo](http://clintonhalpin.github.io/watch-view/dist/#/preview)

### Getting started

```
npm i 
```
Run the app with

```
gulp dist
gulp
```

Open your browser at [http://localhost:4000](http://localhost:4000)
